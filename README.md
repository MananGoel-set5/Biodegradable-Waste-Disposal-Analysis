
# Biodegradable Waste Disposal Analysis

## Project Overview
This project analyzes biodegradable waste disposal patterns using machine learning techniques.

The goal is to identify waste generation patterns, cluster households based on waste behavior, and predict total biodegradable waste generation.

## Dataset
The dataset contains 250 household records with 15 features related to waste generation and disposal behavior.

### Features Include
- Household size
- Daily food waste
- Garden waste
- Paper waste
- Composting practices
- Distance to waste center
- Waste collection frequency
- Awareness score
- Recycling participation rate
- Total biodegradable waste

## Methods Used

### Unsupervised Learning
- K-Means Clustering
- PCA (Principal Component Analysis)

### Supervised Learning
- Gradient Boosting Regressor
- Neural Network (MLP Regressor)

## Evaluation Metrics
- RMSE (Root Mean Squared Error)
- Precision
- Recall
